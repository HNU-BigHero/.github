### 我们是谁

**BigHeroX（超能麓团队）** 是一支长期参与 **RoboCup 机器人足球赛事（中型组）** 的学生团队，隶属于湖南大学机器人工作室。

我们的目标不复杂：
- 把机器人做得更聪明
- 把系统跑得更稳定  
- 把比赛赢下来

BigHeroX 这个名字来自电影《超能陆战队》。不是为了情怀，是因为这个名字够直白。

## 🎯 名字的含义

<div align="center">

| 符号 | 含义 |
|:---:|:---|
| **BigHero** | 团队协作，而不是单点英雄<br/>每一台机器人都不是配角 |
| **X** | 代表未知与变量<br/>我们接受不确定性，也制造不确定性 |
| **变量团队** | 作为新生力量，我们更像赛场上的"扰动项"<br/>而不是背景噪声 |
